X-CUBE-AWS Projects

The application samples are located under STM32CubeExpansion_Cloud_AWS_VX.Y.Z/Projects/Applications, and all of them have the same structure:

  • /[CMx]/Inc header files of the application.
  • [/config_files/] user configuration files of FreeRTOS and its dependencies.
  • /[CMx]/Src source code of the application.
  • /EWARM, /MDK-ARM, and /STM32CubeIDE pre-configured projects for each toolchain.
  • readme.txt describing the application behavior and the environment required to run the example.

Note: refer to section “Development Toolchains and Compilers” and “Supported Devices and boards” of the Firmware package release notes to know about the SW/HW environment used for the Firmware development and validation.
